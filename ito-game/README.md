# ito ボードゲーム アプリ

スマートフォン対応の「ito」ボードゲームWebアプリケーションです。

## ゲームについて

「ito」は、各プレイヤーが1〜100の数字カードを受け取り、テーマに沿って自分の数字の大きさを表現するコミュニケーションゲームです。プレイヤーは相談しながら、小さい順にカードを出していくことを目指します。

## 機能

- 📱 スマートフォン完全対応（レスポンシブデザイン）
- 👥 3〜8人のプレイヤーに対応
- 🎲 1〜100のランダムな数字を自動配布
- 🎨 カスタムテーマ設定（またはランダム選択）
- ✨ 美しいアニメーション効果
- 📊 ゲーム結果の自動判定
- 🔄 簡単なゲームリセット機能

## 使い方

### 1. ファイルの準備

以下の3つのファイルを同じフォルダに配置してください：
- `index.html`
- `styles.css`
- `script.js`

### 2. アプリの起動

#### スマートフォンで使う場合

**方法1: ローカルサーバーを起動**
```bash
# Pythonがインストールされている場合
python -m http.server 8000

# または Node.jsがインストールされている場合
npx http-server -p 8000
```

その後、同じWi-Fi内のスマートフォンから `http://[PCのIPアドレス]:8000` にアクセス

**方法2: 直接ファイルを開く**
- `index.html` をダブルクリックしてブラウザで開く
- スマートフォンに転送してブラウザで開く

#### PCで使う場合
`index.html` をブラウザで開くだけでプレイできます。

### 3. ゲームの流れ

1. **ゲーム設定**
   - プレイヤー数を選択（3〜8人）
   - テーマを入力（空欄の場合はランダムテーマ）
   - 「ゲーム開始」をタップ

2. **数字の配布**
   - 各プレイヤーに順番に端末を渡す
   - 「数字を見る」をタップして自分の数字を確認
   - 他のプレイヤーに見られないように注意！
   - 「次のプレイヤーへ」をタップ

3. **ゲームプレイ**
   - テーマに沿って自分の数字を表現し合う
   - 相談して小さい順にカードを出していく
   - プレイヤーを選択して「カードを公開」

4. **結果確認**
   - 全員のカードが出されたら結果を表示
   - 小さい順に並んでいれば成功！

## カスタマイズ

### 数字の範囲を変更
`script.js` の `generateUniqueNumbers` 関数内の以下の行を編集：
```javascript
const num = Math.floor(Math.random() * 100) + 1; // 1-100の範囲
```

### カラーテーマの変更
`styles.css` の `:root` セクションで色変数を編集：
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* その他の色... */
}
```
